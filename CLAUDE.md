# 血肉苦短研讨班 - Claude Code 规范

> 适用工具: Claude Code

## 仓库信息

- 地址: https://github.com/BoHuYeShan/flesh-is-weak-seminar
- 框架: VitePress + Vue 3
- 部署: GitHub Pages

## 目录结构

```
.vitepress/          # VitePress 配置和主题
*.md                 # 页面文件
.github/workflows/   # GitHub Actions
```

## 禁止操作

- 删除任何 .md 文件
- 删除 .vitepress/ 目录
- 删除 .github/ 目录
- 执行 git push --force

## 本地开发

```bash
npm install
npm run dev
```
