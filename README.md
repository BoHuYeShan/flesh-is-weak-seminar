# 血肉苦短研讨班 👽

群友讨论与分享。

## 在线访问

https://bohuyeshan.github.io/flesh-is-weak-seminar/

## 功能

- 📰 新闻 - 群友分享的最新资讯
- 🔧 小工具 - 群友开发的实用工具
- 💬 讨论 - 热门话题与交流
- 👥 贡献者 - 活跃群友展示

## 如何参与

### 发起讨论

1. 访问 https://github.com/BoHuYeShan/flesh-is-weak-seminar/discussions
2. 选择分类（新闻/小工具/讨论）
3. 发起新讨论

### 提交代码

1. Fork 仓库
2. 创建分支
3. 提交 PR

## 本地开发

```bash
npm install
npm run dev
```

## 技术栈

- VitePress
- Vue 3
- GitHub Discussions API
- GitHub Pages
